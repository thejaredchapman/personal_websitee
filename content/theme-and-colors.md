# Theme & Colors

## Theme Modes
1. **Light** (default)
2. **Dark** (toggled via `data-theme="dark"`, persisted to localStorage key `theme-preference`)

## Color Presets (8 + custom)
1. **Orange** (default) -- #f97316
2. **Blue** -- #3b82f6
3. **Green** -- #22c55e
4. **Purple** -- #a855f7
5. **Pink** -- #ec4899
6. **Teal** -- #14b8a6
7. **Red** -- #ef4444
8. **Yellow** -- #eab308
9. **Custom** -- Color wheel picker with HSL/RGB/HEX inputs, generates a full 50-900 palette from any hex color

Each preset includes a complete 50-900 shade scale.
Persisted to localStorage keys: `accent-color-preference` and `accent-custom-color`.
